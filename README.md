# BlockScore

On-chain wallet reputation scores powered by ML.

## Features
- Analyze Solana wallet history
- Score based on: age, activity, DeFi usage, token diversity
- Shareable score cards for X

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Deploy to Vercel or any Next.js hosting.
